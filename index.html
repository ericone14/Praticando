Questionário

Questão 1: Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo
R: O servidor web entrega arquivos estáticos de um site para o usuário. Ex: Apache.

Questão 2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?
R: A principal característica é trabalhar de forma eficiente com conexões simultâneas, utilizando uma quantidade mínima de recursos do servidor. Por esse motivo é utilizado em projetos de alto desempenho.

Questão 3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?
R: HTTP permite a troca de informações entre cliente e servidor, mas não criptografa os dados, diferente do HTTPS que utiliza criptografia SSL/TLS para proteger os dados transmitidos, garantindo confidencialidade e integridade da informação.

Questão 4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?
R: As portas funcionam como pontos de acesso lógicos que permitem que diferentes serviços e aplicações se comuniquem entre si. A porta 443 é usada para o HTTPS. Ela é fundamental para a segurança na web.

Questão 5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.
R: Um endereço IP funciona como o "endereço residencial" de um dispositivo na rede. Ele permite que os dados enviados pela rede cheguem ao destino correto. É importante para identificação de dispositivos, roteamento de dados, comunicação entre dispositivos, geolocalização, segurança e controle.

Questão 6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.
R: É um servidor virtual com mais controle e desempenho. Indicado quando precisa de hospedagem de vários sites com tráfego alto.

Questão 7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.
R: Facilidade de uso, segurança embutida e configuração automática de HTTPS.

Questão 8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.
R: O Kestrel é o mais indicado por conta do seu alto desempenho, integração nativa com o .NET e otimizado para C#.

Questão 9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web
R: O cliente (como um navegador) faz um pedido a um servidor, que processa esse pedido e envia uma resposta com os dados requisitados (como uma página web). Essa troca acontece por meio do protocolo HTTP, e a conexão termina após cada interação.

Questão 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado
R: O Apache Tomcat é um servidor de aplicações open-source projetado especificamente para executar aplicações web baseadas em Java

Questão 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais
R: O servidor web mostra o que já está pronto, o servidor de aplicação cria o que for necessário. O servidor web lida com requisições HTTP e entrega conteúdo estático. O servidor de aplicação executa lógica de negócio e gera conteúdo dinâmico.

Questão 12: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.
R: O servidor web entrega conteúdo estático e o servidor de aplicação gera conteúdo dinâmico.

Questão 13: Defina o conceito de domínio na internet e sua relação com os endereços IP.
R:O domínio na internet é o nome legível que usamos para acessar sites, como google.com. Ele serve como um atalho amigável para um endereço IP, que é a identificação numérica real de um servidor na internet.

Questão 14: Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?
R: Escalabilidade, alta disponibilidade, redução de custos, acesso remoto e segurança.

Questão 15: Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.
R: Apache Tomcat suporta Java e Node.js suporta JavaScript.

Questão 16: Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?
R: O HTTPS é essencial por questões de segurança, privacidade e confiabilidade. O HTTP apresenta riscos como roubo de informações, falsificação de sites e vulnerabilidade a scripts maliciosos.

Questão 17: Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.
R: Em uma loja virtual, por conta de recursos dedicados, melhor desempenho, mais controle e segurança. Na hospedagem compartilhada o site poderia ficar lento em horários de pico.

Questão 18: Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.
R: IP identifica unicamente um dispositivo em uma rede, permite que os dados cheguem ao dispositivo correto em uma rede. Porta identifica qual serviço ou aplicação dentro do dispositivo deve receber os dados.

Questão 19: Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?
R: Segurança dos dados, confiança do usuário. Sem HTTPS o servidor web expõe os dados dos usuários a riscos e pode ser bloqueado por navegadores ou firewalls.

Questão 20: Cite dois exemplos de servidores web e explique suas principais características.
R: Apache: Open Source, suporta múltiplos sistemas operacionais, altamente configurável compatível com várias linguagens de programação via integração. Nginx: foco em alto desempenho e baixo consumo de recursos, bom para lidar com grande volume de conexões simultâneas.
